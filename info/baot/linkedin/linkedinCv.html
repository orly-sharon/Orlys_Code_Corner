עריכת פרופיל לינקדאין: ניסיון מקצועי ומיומנויות

💼 כתיבת ניסיון מקצועי (Experience)

בפרק הניסיון המקצועי, המטרה היא לייצר תיאור רחב ומפורט של התפקידים השונים (תזכורת: יש לכתוב באנגלית בלבד). מומלץ להציג את תחומי האחריות וההישגים בנקודות (Bullet Points) קצרות וענייניות. בעת העדכון, הקפידי על הדגשים הבאים:

בחירת מוסד ולוגו תואם

כאשר אתן מוסיפות מקום עבודה, לימודים או התנדבות, ודאו כי בחרתן את המוסד הנכון מתוך הרשימה הנגללת, וכי הלוגו של המוסד שנבחר תואם את המקום בו למדתן/עבדתן (ישנן לא מעט חברות עם שמות דומים מאוד).

הקפדה על זמנים מדויקים

מקומות עבודה או לימודים בעבר יש לכתוב בזמן עבר (Past Tense). מקומות נוכחיים יש לכתוב בזמן הווה (Present Tense). הקפידו על הזמנים הנכונים בכל עדכון שאתן מבצעות.

מניעת חפיפת תאריכים

שימו לב שאין חפיפה בין תאריכי העבודות השונות שלכן לאורך ציר הזמן, אלא אם כן עבדתן בשני מקומות במקביל באותה תקופה בפועל.

שימוש ב-Buzz Words

הקפידו לפרט ככל שניתן ולשלב את מילות המפתח המקצועיות ומונחי הטכנולוגיה הנכונים (Tech Stack) הרלוונטיים לתחומכן, בצורה ממוקדת וברורה.

תיאור קצר

בשורה . נקודה הראשונה הקפידי רשום תיאור קצר של תחום העבודה ומה היה התפקיד. (גם כאן הקפידי על זמנים)

לדוגמה: Developed an insurance-related information system in JAVA (using Spring Boot and Maven)

דוגמה למשפטים קצרים, ענייניים ושילוב Buzz Words (Example):

כתיבה עבור תפקיד בעבר (זמן עבר):

• Developed an insurance-related information system in JAVA (using Spring Boot and Maven).
• Wrote and ran SQL queries (MSSQL) using Hibernate.
• Tech Stack: AWS services including EC2, S3, IAM, DynamoDB, and Lambda.
• Worked with Agile methodology.
• Worked with JetBrains tools, including IntelliJ IDEA and TeamCity.
• Worked with Atlassian tools, including Bitbucket and Jira.

כתיבה עבור תפקיד נוכחי (זמן הווה):

• Developing an insurance-related information system in JAVA (using Spring Boot and Maven).
• Writing and running SQL queries (MSSQL) using Hibernate.
• Tech Stack: AWS services including EC2, S3, IAM, DynamoDB, and Lambda.
• Working with Agile methodology.
• Working with JetBrains tools, including IntelliJ IDEA and TeamCity.
• Working with Atlassian tools, including Bitbucket and Jira.

🎯 מיומנויות (Skills)

שימי לב: אם אין לך מיומנויות רשומות בפרופיל, אזור ה-Skills לא יופיע כלל לגולשים חיצוניים.

➕ כיצד להוסיף את אזור המיומנויות ומיומנויות חדשות:

  1. לחצי על אייקון "אני" (Me) בראש דף הבית של לינקדאין, ולאחר מכן על הצג פרופיל (View Profile).
  2. לחצי על כפתור Add profile section (הוספת חלק לפרופיל) שנמצא באזור המבוא שלך.
  3. לחצי על קטגוריית Core ובחרי ב-Add skills מתוך התפריט הנפתח.
  4. בחלון הקופץ שנפתח, תוכלי:
    • הקלידי שם של מיומנות ובחרי אותה מתוך הרשימה הנגללת שמופיעה, או בחרי מיומנות מתוך הרשימה המוצעת (Suggested) המבוססת על הפרופיל שלך. ברגע שתבחרי, היא תתווסף אוטומטית לרשימה.
    • סמני בתיבות הסימון (Checkboxes) היכן הפעלת והשתמשת במיומנות זו בפועל (שיוך לעבודות/לימודים).
    • סמני את תיבת הסימון במידה ואת מעוניינת לעקוב אחר עמוד המאמרים השיתופיים של מיומנות זו.
  5. לחצי על Save (שמור).
  6. חזרי על השלבים שלעיל כדי להוסיף מיומנויות נוספות לפרופיל שלך.

❌ כיצד להסיר מיומנויות קיימות:

  1. לחצי על אייקון "אני" (Me) בראש דף הבית של לינקדאין, ולאחר מכן על הצג פרופיל (View Profile).
  2. גללי מטה אל אזור ה-Skills ולחצי על אייקון העריכה (עיפרון).
  3. בעמוד המיומנויות שנקרא, לחצי על אייקון העריכה (עיפרון) מימין למיומנות שברצונך למחוק.
  4. לחצי על Delete skill בפינה השמאלית התחתונה, ולאחר מכן אשרי שוב בלחיצה על Delete.

ℹ️ במידה ונתקלת בבעיה כלשהי, מומלץ לפנות למדריך הרשמי של לינקדאין להוספה והסרה של מיומנויות.
